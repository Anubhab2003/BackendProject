## Installation

Install my-project with npm

```bash
  git clone https://github.com/Anubhab2003/BackendProject.git

  cd BackendProject

  npm init

  npm run dev
```
    
## Running Tests

To run tests, run the following command

```bash
  npm run dev
```

To test this project use Postman




## Tech Stack

**Client:** PostMan

**Server:** Node, Express, MongoDB, Multer
